# Report on L293D Motor Driver IC

## Abstract
The L293D motor driver IC is used to control DC motors and other inductive loads using microcontrollers. Since microcontrollers provide low current, the L293D acts as an interface that supplies the required current to drive motors. This report explains the working of the L293D motor driver IC, its H-bridge configuration, PWM-based speed control, and its main applications.

---

## 1. Introduction
The L293D is a motor driver integrated circuit used to control the direction and speed of DC motors. Microcontrollers alone cannot drive motors because they cannot supply sufficient current. The L293D solves this problem by acting as a current amplifier between the control circuit and the motor.

The IC can control two DC motors simultaneously and is commonly used in robotics and embedded system projects.

---

## 2. Pin Description
The L293D IC has 16 pins. Important pins include:

| Pin Type | Function |
|--------|---------|
| Vcc1 | Logic power supply (5V) |
| Vcc2 | Motor power supply |
| GND | Ground |
| Input Pins | Control signals from microcontroller |
| Output Pins | Connected to motor |
| Enable Pins | Used to enable motor driver and apply PWM |

---

## 3. H-Bridge Concept
The L293D uses an **H-bridge circuit** to control the direction of a motor.

An H-bridge allows the current to flow in two directions through the motor, which enables forward and reverse rotation.

| Input 1 | Input 2 | Motor Action |
|--------|--------|--------------|
| 1 | 0 | Forward |
| 0 | 1 | Reverse |
| 0 | 0 | Stop |
| 1 | 1 | Brake |

---

## 4. PWM (Pulse Width Modulation)
PWM is used to control the speed of a motor. Instead of changing the voltage, the motor is switched on and off rapidly.

The speed depends on the **duty cycle** of the signal.

Example:

| Duty Cycle | Motor Speed |
|-----------|-------------|
| 25% | Low speed |
| 50% | Medium speed |
| 75% | High speed |
| 100% | Maximum speed |

PWM signals are usually applied to the **Enable pins** of the L293D.

---

## 5. Electrical Specifications

| Parameter | Value |
|-----------|-------|
| Logic Voltage | 4.5V – 7V |
| Motor Voltage | Up to 36V |
| Output Current | 600 mA per channel |
| Peak Current | 1.2 A |

---

## 6. Applications
The L293D motor driver IC is widely used in:

- Robotics
- Line follower robots
- DC motor control systems
- Stepper motor control
- Embedded system projects

---

## 7. Conclusion
The L293D motor driver IC is a simple and effective device for controlling DC motors. It uses an H-bridge circuit to control motor direction and PWM signals to control motor speed. Because of its easy interfacing and built-in protection, it is widely used in robotics and embedded system applications.
